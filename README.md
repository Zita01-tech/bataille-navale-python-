# Bataille Navale – Python

## Description
Ce projet est une implémentation du jeu de **bataille navale** en Python, jouable en ligne de commande.  
Il permet à un joueur d’affronter l’ordinateur dans une partie complète respectant les règles classiques du jeu.

Le projet met en œuvre des notions de **programmation structurée**, de **logique algorithmique** et de **gestion de données** sous forme de grilles.

---

##  Fonctionnalités
- Création de grilles de jeu (joueur et ordinateur)
- Placement manuel des bateaux pour le joueur
- Placement aléatoire des bateaux pour l’ordinateur
- Gestion des tirs (touché, coulé, à l’eau)
- Détection de la fin de partie (victoire / défaite)
- Affichage partiel de la grille adverse (brouillard de guerre)
- Commandes utilisateur (`AFFICHER`, `STOP`)

---

##  Concepts abordés
- Structures de données (listes, matrices)
- Fonctions et modularité du code
- Boucles et conditions
- Génération aléatoire
- Interaction utilisateur via le terminal
- Gestion des erreurs et validation des entrées

---

##  Technologies utilisées
- **Langage** : Python  
- **Bibliothèques** : `random` (standard)
